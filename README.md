## Descripción del Proyecto
Este proyecto es una solución web profesional diseñada para **Imagenia**, un estudio de fotografía y servicios de impresión. El objetivo principal fue digitalizar el negocio local con una interfaz moderna, rápida y optimizada para la conversión de clientes.

### Características Principales
* **Performance Extremo:** Desarrollado con Next.js para tiempos de carga instantáneos.
* **Diseño Responsive:** Experiencia perfecta en móviles, tablets y ordenadores.
* **SEO Optimizado:** Estructura técnica preparada para posicionar en Google localmente.
* **UI/UX Moderna:** Interfaz limpia que prioriza la identidad visual del estudio.

### Stack Tecnológico
* **Framework:** Next.js 14+ (App Router)
* **Estilos:** Tailwind CSS
* **Componentes:** Shadcn/UI & Lucide Icons
* **Despliegue:** Vercel (CI/CD)

## Project Overview
This project is a professional web solution designed for **Imagenia**, a photography and printing studio. The main goal was to digitize the local business with a modern, fast, and conversion-optimized interface.

### Key Features
* **Extreme Performance:** Built with Next.js for near-instant loading times.
* **Fully Responsive:** Seamless experience across mobile, tablet, and desktop devices.
* **SEO Optimized:** Technical structure ready for local Google ranking.
* **Modern UI/UX:** A clean interface that highlights the studio's visual identity.

### Tech Stack
* **Framework:** Next.js 14+ (App Router)
* **Styling:** Tailwind CSS
* **Components:** Shadcn/UI & Lucide Icons
* **Deployment:** Vercel (CI/CD)
